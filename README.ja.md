# gltfbeauty

[gltfbeauty](https://code4fukui.github.io/gltfbeauty/)は、大きなGLTF/GLBファイルをウェブ公開用に縮小するツールです（まとめて変換対応、サーバーに送信されません）。

## デモ
[gltfbeauty](https://code4fukui.github.io/gltfbeauty/)で試すことができます。

## 機能
- GLTF/GLBファイルのテクスチャサイズを圧縮
- 出力ファイルはGLTFまたはGLB形式を選択可能
- 複数ファイルの一括処理に対応

## 使い方
ブラウザで[gltfbeauty](https://code4fukui.github.io/gltfbeauty/)を開き、GLTF/GLBファイルをドロップするか、フォルダを選択して処理を行います。
