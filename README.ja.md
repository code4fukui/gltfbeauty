# gltfbeauty

[gltfbeauty](https://code4fukui.github.io/gltfbeauty/)は、大きなGLTF/GLBファイルのテクスチャサイズを縮小し、ウェブ公開に適した形式に変換するツールです。複数ファイルの一括処理や、ブラウザ上での完全なオフラインでの変換が可能です。

## 機能
- GLTF/GLBファイルのテクスチャサイズを圧縮
- 出力ファイルの形式をGLTFまたはGLBから選択可能
- 複数ファイルの一括処理に対応

## 使い方
1. ブラウザで[gltfbeauty](https://code4fukui.github.io/gltfbeauty/)を開きます。
2. GLTF/GLBファイルをドラッグ&ドロップするか、フォルダを選択して処理を開始します。
3. テクスチャサイズ、品質、出力形式を調整します。
4. 変換されたファイルはZIPファイルでダウンロードできます。
